# Home Screen Widgets with React Native  
**React India 2023 — October 2023, Goa Velha, India**

At React India 2023, I shared implementation strategies and developer considerations for bringing home screen widgets to React Native applications. This talk was grounded in both technical detail and the larger context of user engagement and platform capabilities.

## Talk Overview

The session focused on bridging the gap between JavaScript and native layers to unlock widget support in mobile apps built with React Native.

**Topics Covered**
- Understanding what home screen widgets can offer for app engagement
- Limitations of React Native's core for widget rendering
- Overview of Native Modules and how to interface with iOS and Android APIs
- Structuring widget-related code for maintainability and cross-platform reuse
- Real-world examples of widget functionality: toggles, previews, status indicators
- Future direction of widget APIs and ecosystem gaps
- Developer experience concerns: debugging, hot reload limitations, CI challenges

## Key Points
- Widgets are a powerful but underutilized surface in React Native apps
- Implementing them requires writing and maintaining native code, bridged via JS
- Careful architectural decisions are necessary to keep the codebase clean and modular
- Investing in DX tools and abstractions can make widget development repeatable and scalable

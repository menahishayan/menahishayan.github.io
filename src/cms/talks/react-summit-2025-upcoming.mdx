# React Beyond the Browser

**React Summit 2025 — Amsterdam, The Netherlands**

At React Summit 2025, I spoke about applying React in environments that go beyond traditional web browsers. The session introduced new frontiers for React, ranging from enterprise productivity tools to hardware-powered UI systems.

## Talk Overview

The talk was split across two themes: productivity plugins and embedded systems.

**Office 365 Plugins**

- Discussed integration with Office JavaScript APIs
- Demonstrated how React can be used to develop plugins for Word and Outlook
- Showcased a plugin prototype that leverages OpenAI inside Office 365

**Embedded Systems**

- Walked through rendering React on SPI displays using Raspberry Pi
- Shared the technical process of framebuffer mirroring with React on a 320x240 LCD
- Covered system performance, rendering latency, and constraints of embedded UIs

**Additional Contexts**

- Highlighted how React can be rendered inside AR/VR engines
- Touched on potential applications within LVGL and automotive infotainment systems

## Key Points

- React's component model can adapt to non-browser environments
- With the right rendering layer, React can serve as a universal UI language
- Developers can unlock use cases in productivity, hardware, and immersive computing

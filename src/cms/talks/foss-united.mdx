# HomeScript — Command Line Control of Your Home

**FOSS United — December 2023, Bengaluru, India**

At FOSS United 2023, I presented _HomeScript_, a command-line tool built to control HomeKit accessories without the Apple Home app. The tool was designed to bridge Python and HomeBridge APIs, making smart home automation accessible from the terminal.

## Talk Overview

The session walked through the motivations behind the project, the underlying architecture, and the unique challenges of creating a CLI-based automation system for smart homes.

**Topics Covered**

- The motivation for CLI-based smart home control
- Architecture of HomeScript and how it integrates with HomeBridge
- Real-time demos: toggling accessories, adjusting lights, setting groups
- Implementation of EasyMatch for fuzzy accessory name matching
- Extending HomeScript for advanced control: brightness, hue, saturation
- Automating routines using cron and shell scripts
- Packaging the project as a Python module and building an API layer
- Security implications and the requirement to run HomeBridge in insecure mode

## Key Points

- HomeScript enables full smart home control via terminal or automation scripts
- It supports RGB lighting, grouped device actions, and a full API interface
- The project is lightweight, open source, and actively maintained
- Designed for extensibility, it's suited for developers, power users, and tinkerers
